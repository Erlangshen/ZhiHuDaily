# ZhiHuDaily
这个开源是一个简单的知乎日报的编写，主要目的是练习MaterialDesign的相关知识、以及RecycleView、CardView的使用。<br>
 ![](https://github.com/gcgongchao/ZhiHuDaily/blob/master/gif/zhihu.gif)<br>
